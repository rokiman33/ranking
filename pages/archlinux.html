11 Arch Linux

Arch Linux es una distribución Linux para computadoras x86-644​ orientada a usuarios avanzados. Se compone en su mayor parte de software libre y de código abierto (FOSS)5​ y apoya la participación comunitaria.6​ El enfoque de diseño del equipo de desarrollo, sigue el principio KISS como línea general, y se centra en la elegancia, exactitud, minimalismo y simplicidad, y espera que el usuario esté dispuesto a realizar un esfuerzo por entender el funcionamiento del sistema. El gestor de paquetes escrito específicamente para Arch, llamado Pacman, se usa para instalar, eliminar y actualizar paquetes.
